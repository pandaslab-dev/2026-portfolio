# Andy Mills Portfolio

A personal portfolio site that highlights selected software projects through a playful, animated single-page experience.

## Why I built it

I built this site to create a portfolio that feels more personal than a standard resume page. I wanted a place to showcase projects I care about, experiment with visual presentation and motion, and practice building a polished front-end experience that still stays simple to maintain.

## Tech stack

- TypeScript
- React 19
- Next.js 16 App Router
- Tailwind CSS v4
- Motion
- Netlify

## Current features

- Animated hero section with custom visual styling and motion
- Project showcase cards driven by a central portfolio data file
- External links for live projects and GitHub repositories
- Contact section with direct email access
- Static export setup for simple hosting and deployment

## What I learned

- How to structure a small Next.js App Router project so content stays easy to update
- How motion and typography choices affect the feel of a portfolio site
- How to balance visual personality with responsive layout and maintainable components
- How to prepare a front-end project for static deployment

## What I would improve next

- Add tests for key UI behavior and content rendering
- Expand project entries into deeper case studies
- Keep refining the mobile layout and spacing
- Add a lightweight content workflow for easier future updates

## Running locally

```bash
npm install
npm run dev
```
