# Andy Mills Portfolio

My attempt at an actually professional portfolio site while still trying to maintain my personality and character.


## Tech stack

- TypeScript
- React 19
- Next.js 16 App Router
- Tailwind CSS v4
- Motion
- Netlify for static deploy :)

## Current features

- Animated hero section with custom visual styling and motion
- Project showcase cards driven by a central portfolio data file
- External links for live projects and GitHub repositories
- Contact section with direct email access

## What I learned

- How to structure a small Next.js App Router project so content stays easy to update
- Motion Graphics/Kinetic Typography using Motion
- How to balance visual personality with professional clarity
- How to prepare a front-end project for static deployment
